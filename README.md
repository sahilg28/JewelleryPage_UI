# E-commerce Jewelry UI

A responsive e-commerce product detail page for jewellery, built with React and Tailwind CSS.


## Tech Stack

- React 19
- Vite
- Tailwind CSS
- JavaScript

## Live Demo

[View Live Site](https://sahilg28.github.io/JewelleryPage_UI/)

## Development

```bash
npm install
npm run dev
```

## Deployment

```bash
npm run deploy
```
